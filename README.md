# Woolytrading

## Projektziel
Dieses Projekt dient der Entwicklung einer modernen Homepage für eine Kundin, auf der sie ihre eigenen TCG-Sammelkarten, Booster und verwandte Produkte verkaufen kann.
Der Fokus liegt auf einer übersichtlichen Produktdarstellung und einer soliden technischen Basis für zukünftige Erweiterungen.

## Tech-Stack
 - Frontend: Angular
 - Backend: ASP.NET Core (.NET)
 - Datenbank: geplant (wird später ergänzt)
 - Architektur: Client-Server (REST API)
